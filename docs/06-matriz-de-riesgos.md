# 06 — Matriz de Riesgos

La evaluación fue realizada considerando probabilidad e impacto, expresados según los niveles establecidos en MicroPILAR (L0 a L5).

| Riesgo | Activo asociado | Nivel actual | Nivel objetivo | Estado |
|--------|---------------|--------------|----------------|--------|
| Acceso no autorizado | ACT-SE C-01 / ACT-DBT-01 | L1 | L4 | Pendiente |
| Pérdida de datos | ACT-DBT-01 | L1 | L4 | Pendiente |
| Interrupción del servicio | ACT-SRV-01 / ACT-RMF-01 / ACT-NET-01 | L1–L2 | L4 | Pendiente |
| Vulnerabilidades técnicas | Software Open-RMF / Robots | L1 | L3 | Pendiente |
| Exposición de logs | ACT-DBT-01 | L1 | L3 | Pendiente |

## Conclusiones

El sistema actualmente presenta **niveles bajos de madurez de seguridad (L0-L2)** y deberá alcanzar **niveles superiores (L3-L5)** para cumplir lineamientos ENS + ISO.
