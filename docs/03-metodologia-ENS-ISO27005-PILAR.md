# 03 — Metodología Aplicada

El análisis se realizó utilizando la herramienta **MicroPILAR**, basada en los siguientes estándares:

| Norma / Marco | Propósito |
|---------------|-----------|
| ISO/IEC 27005 | Gestión de riesgos de seguridad de la información |
| ENS (Esquema Nacional de Seguridad) | Cumplimiento en entornos públicos |
| ISO/IEC 27001 | Controles de seguridad |
| RGPD | Cumplimiento de protección de datos |

MicroPILAR permitió el modelado del sistema mediante:

- Identificación de activos
- Clasificación del valor según confidencialidad, integridad y disponibilidad (CIA)
- Asociaciones con posibles amenazas y escenarios
- Evaluación probabilística y cualitativa del riesgo

El archivo original del análisis se encuentra en: PRJ-CLINICA-ROBOTS-2025.mgr
