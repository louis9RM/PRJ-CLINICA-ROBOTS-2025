# 01 — Contexto del Proyecto

El proyecto consiste en la implementación de un sistema de gestión de flota de robots móviles autónomos (AMR) dentro del entorno hospitalario de la Clínica Cayetano Heredia. El sistema utilizará la plataforma **Open-RMF (Robotics Middleware Framework)** como base tecnológica para coordinar varios robots heterogéneos.

El objetivo es realizar transporte seguro, trazable y automatizado de:

- Medicamentos
- Muestras biológicas
- Insumos médicos
- Residuos sanitarios

Este sistema busca mejorar la eficiencia logística hospitalaria, reducir tiempos críticos y disminuir los riesgos humanos asociados a la manipulación de recursos clínicos.

Adicionalmente, se emplea la herramienta **MicroPILAR** como soporte metodológico para la evaluación de riesgos basada en ENS e ISO 27005, permitiendo:

- Identificar activos críticos
- Evaluar amenazas y vulnerabilidades
- Determinar niveles de riesgo
- Definir planes de tratamiento
- Mantener evidencia documental formal
